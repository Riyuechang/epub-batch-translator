# 術語表檔案
一共有兩種類型，預設和自動匹配。

**預設術語表**
如果 epub 沒有與之匹配的術語表，則使用預設術語表。
預設術語表檔名如下。
```
default.json
```

**自動匹配術語表**
如字面意思，會自動匹配與對應的 epub 。
匹配規則如下。
```
# 用書名命名術語表檔
書名.json

# 自動匹配包含書名的 epub 。
書名 1.epub
書名 2.epub
書名 3.epub

# 就算格式亂七八糟，只要包含對應的書名就能匹配。
【特點】書名-4.epub
【作者x插畫師】書名_5.epub
【☆OO出版社限定☆】書名6.epub
```
當然也能一個 epub 一個術語表檔。
